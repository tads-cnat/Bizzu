Descrição do repositório

Este repositório contém a implementação de uma Pilha Dinâmica em Java. O projeto inclui uma classe PilhaDinamica que permite empilhar, desempilhar e verificar o elemento no topo, além de métodos auxiliares para manipulação da pilha, como checar se está vazia. A pilha é construída com uma estrutura de nós encadeados, permitindo redimensionamento dinâmico sem limite fixo de tamanho. Ideal para estudar estruturas de dados e conceitos de alocação dinâmica em Java.
