# 1. Cronograma de Marcos

Um cronograma de marcos é uma forma de planejamento que destaca os principais eventos, entregas ou fases de um projeto — chamados de marcos. O cronograma de marcos foca apenas nos pontos-chave de progresso, como o início ou término de etapas importantes. Ele é usado para acompanhar o avanço do projeto de forma resumida, facilitando a comunicação entre a equipe e as partes interessadas.
