# Regras de negócio

**Histórico de Alterações:**
| Versão | Autor | Data |
|--------|-------|------|
| 1.0 | Ana Maria | 02 de agosto de 2025 |

| N°  | Descrição                                                             | Contexto                                                                                                                      | Ação do sistema                                                                                                                                       |
| --- | --------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| 001 | Revisar e aceitar solicitações de mudança de papel para moderador     | Apenas moderadores podem revisar e aceitar solicitações de internautas que desejam se tornar moderadores                      | Mostrar aba de "Solitações" no perfil de moderadores. Internautas não visualizam essa opção.                                                          |
| 002 | Revisar denúncias de conteúdos (comentários, postagens, repositórios) | Apenas moderadores podem revisar e aceitar denúncias de conteúdos                                                             | Mostrar aba de "Denúncias" no perfil de moderadores. Caso ele aprove a denúncia o conteúdo é removido caso não seja aprovada a denúncia é descartada. |
| 003 | Solicitação de mudança de papel para moderador                        | Somente internautas podem solicitar alteração de papel                                                                        | Mostrar botão "Solicitar mudança de papel" apenas no perfil de internautas.                                                                           |
| 004 | Realização de denúncias                                               | Usuários autenticados podem denunciar conteúdos                                                                               | Visitantes podem apenas visualizar postagens, sem opções de denúncia.                                                                                 |
| 005 | Interações de visitantes                                              | Visitantes não podem interagir com o feed (curtir, comentar, visitar perfil)                                                  | Exibir aviso ao tentar interagir: "Faça o login para aproveitar integralmente o bizzu".                                                               |
| 006 | Manutenção das categorias da plataforma                               | Somente moderadores podem editar/manter categorias                                                                            | Exibir aba de categorias somente para moderadores.                                                                                                    |
| 007 | Denúncia de conteúdo próprio                                          | Usuário não pode denunciar conteúdo que ele mesmo publicou                                                                    | Substituir botão de denúncia por botões de edição e exclusão no conteúdo próprio.                                                                     |
| 008 | Processo de cadastro e definição de papel inicial                     | Visitantes tornam-se internautas ao concluir cadastro. Ele pode se tornar um moderador somente após aprovação de solicitação. | Cadastro inicial define papel como internauta; mudança para moderador requer aprovação de outro moderador.                                            |
| 009 | Verificação de unicidade de username                                  | Username deve ser único no momento do cadastro                                                                                | Exibir aviso de erro e bloquear cadastro se username já existir.                                                                                      |
| 010 | Alteração de perfil                                                   | Apenas usuários autenticados podem atualizar dados do seu próprio perfil                                                      | Outros usuário não podem fazer modificações em perfis que não lhe pertençam.                                                                          |
| 011 | Criação de postagens e comentários                                    | Apenas usuários autenticados (internautas ou moderadores) podem postar e comentar                                             | Visitantes não têm acesso às funções de criar postagens ou comentar.                                                                                  |
| 012 | Visualização de perfis de outros usuários                             | Somente usuários autenticados podem visualizar perfis completos de outros usuários                                            | Visitantes visualizam apenas conteúdo público, sem acesso a detalhes de perfis.                                                                       |
